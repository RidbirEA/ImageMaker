Данная программа получает на вход картинку и создает текстовый файл с этой картинкой, состоящей из символов.

Программа работает на основе дизеринга Флойда – Стейнберга (https://en.wikipedia.org/wiki/Floyd%E2%80%93Steinberg_dithering)

Весь код написан на Python. Также были задействованы модули: OpenCV(cv2) для перевода изображения в массив, Tkinter для создания 
простейшено графического интерфейса, Numba для ускорения вычислений.

После запуска программы необходимо ввести путь к картинке (относительный или абсолютный). Пример: img/image.jpg
